Welcome to the API!

How to use this API:

  • GET /transactions/:id : Get a transaction by ID. Replace ':id' with the ID of the transaction you want to retrieve.
  • POST /transactions : Create a new transaction. The body should be a JSON object with 'userId', 'accountId', 'categoryId', 'amount', 'transactionDate', 'description', and 'type' keys.
  • PUT /transactions/:id : Update an existing transaction. Replace ':id' with the ID of the transaction you want to update. The body should be a JSON object with the 'accountId', 'categoryId', 'amount', 'transactionDate', 'description', and 'type' keys you want to update.
  • DELETE /transactions/:id : Delete a transaction. Replace ':id' with the ID of the transaction you want to delete.
  • GET /transactions/uid/:uid : Retrieve transactions by user ID. Replace ':uid' with the user ID to retrieve transactions for.
